Excel数据查询系统

高效、安全的数据查询与管理平台

项目简介

Excel数据查询系统是一个基于Web的应用程序，用于管理和查询Excel格式的数据文件。系统支持多用户管理、数据搜索、批量导入导出等功能。

智能搜索

支持多关键词搜索，快速定位所需数据。可同时搜索多个Excel文件内容。

安全可靠

多重安全验证，保护您的数据安全。支持用户权限管理和操作日志记录。

响应式设计

适配各种设备，随时随地访问。完美支持PC、平板和手机端。

系统功能

功能模块 功能描述 支持格式
文件管理 上传、下载、删除Excel文件 .xls, .xlsx, .csv
数据查询 多关键词搜索，模糊匹配 所有文本字段
用户管理 添加、编辑、删除用户账户 -
批量操作 批量导入用户，批量审核 .csv, .xlsx

安装指南

  1. 确保服务器环境满足要求：
    • PHP 7.4+
    • MySQL 5.7+
    • Apache/Nginx
  2. 克隆项目仓库：
    git clone https://github.com/your-repo/excel-query-system.git
  3. 安装依赖：
    composer install
  4. 导入数据库：
    mysql -u username -p database_name < sql/schema.sql
  5. 配置环境变量：
    cp .env.example .env
    vi .env
